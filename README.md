# MAC 完整自動化工具 使用說明

## 功能介紹

上傳含「号段分配」工作表的 Excel，自動產生「工廠使用」工作表並下載。

---

## 處理流程

```
上傳 Excel
    ↓
① 讀取「号段分配」工作表
    ↓
② 整理 MAC → mac1~mac4（每 4 筆一組）
    ↓
③ 建立「工廠使用」工作表
    ↓
④ 填入 IMEI + mac1~mac4
    ↓
⑤ 下載 Excel
```

---

## 輸入格式

Excel 需包含名為「号段分配」的工作表：

| 欄位 | 內容 |
|------|------|
| A 欄 | IMEI（第一列為標題） |
| B 欄 | MAC（第一列為標題） |

---

## 輸出格式

新增「工廠使用」工作表，欄位如下：

| 欄 | 標題 | 內容 |
|----|------|------|
| A | IMEI | 逐列對應 IMEI |
| B | mac1 | 第 1、5、9… 筆 MAC |
| C | mac2 | 第 2、6、10… 筆 MAC |
| D | mac3 | 第 3、7、11… 筆 MAC |
| E | mac4 | 第 4、8、12… 筆 MAC |
| F | pcba sn(按照Thinkstart規則生成) | 空白 |
| G | 組裝sn(按照Thinkstart規則生成) | 空白 |

MAC 每 4 筆為一組橫排：

```
第 1~4 筆  →  第 1 列 mac1~mac4
第 5~8 筆  →  第 2 列 mac1~mac4
...
```

---

## 使用步驟

1. 瀏覽器開啟 `MAC完整自動化工具.html`
2. 點擊「開始自動整合處理」
3. 上傳 Excel（`.xlsx` 或 `.xls`）
4. 等待處理完成，確認右側預覽
5. 點擊「下載 Excel」，輸出檔名為 `原檔名_工廠使用.xlsx`

---

## 注意事項

- 「工廠使用」工作表若已存在，會自動覆蓋
- MAC 筆數不足 4 的倍數時，最後一組剩餘欄位保持空白
- F、G 欄由工廠人員手動填入
- 資料處理在瀏覽器本機執行，不上傳伺服器
- 首次開啟需網路連線載入 xlsx.js
